# Home Assistant blueprints

Reusable blueprint versions of selected automation patterns.

## Automation blueprints

- [`automation/robot-mower-weather-gated.yaml`](automation/robot-mower-weather-gated.yaml) — robot mower every second day if weather is dry, with completion notification only for automation-started runs.

## Importing from GitHub

In Home Assistant:

1. Go to **Settings → Automations & scenes → Blueprints**.
2. Select **Import Blueprint**.
3. Paste the GitHub URL for the blueprint file.
4. Create an automation from it and fill in the inputs.

For this repository, the import URL is:

```text
https://github.com/Seigliva/home-assistant-homelab-showcase/blob/main/blueprints/automation/robot-mower-weather-gated.yaml
```
