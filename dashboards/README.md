# Dashboard patterns

These are curated, anonymized dashboard examples from the Home Assistant setup. They are documented as reusable patterns rather than raw Lovelace exports.

## Mobile dashboards

- [Mobile-first main dashboard](mobile-main-dashboard.md) — shared household dashboard with weather, door/garage state, presence, room temperatures, bottom navigation and Broadlink IR heat pump controls.

## Publishing notes

Screenshots are anonymized before publication. Personal photos, names, exact entity IDs and sensitive room/security details are removed or generalized.
