# Integrations used

The exact private setup changes over time, but these are the useful public patterns.

## Home Assistant integrations

- MQTT
- mobile app notifications
- weather entity
- robot mower integration
- vacuum/Roborock-style integrations
- media players
- lights and scenes
- blinds/covers
- energy/power sensors
- calendar/trash-day style sensors

## Homelab integrations

- Unraid Docker monitoring
- Proxmox host awareness
- DNS filtering and local network services
- CLARA/Hermes Agent scheduled jobs
- Telegram notifications

## AI/agent patterns

- local or hosted LLMs for short notification text
- Firecrawl/web research for weekly briefings
- CLARA as a homelab operations assistant
