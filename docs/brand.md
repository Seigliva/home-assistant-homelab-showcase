# ITih brand note

**ITih** means **IT i hjemmet** — Norwegian for "IT in the home".

It also has a small E.T. joke built in: E.T. phones home; ITih automates home.

## Tone

- practical
- nerdy
- useful
- slightly show-off, but not hype
- Norwegian homelab/smart-home personality

## What belongs here

- automations that solve real household problems
- Home Assistant patterns others can copy
- CLARA/Hermes Agent homelab workflows
- screenshots and walkthroughs when private data can be hidden
- lessons learned from running a real self-hosted home
