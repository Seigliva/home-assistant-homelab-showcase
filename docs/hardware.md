# Hardware and devices

This is intentionally generic. The goal is to show the kind of devices used without exposing the exact home layout.

## Device categories

- robot mower
- robot vacuums
- smart blinds/covers
- motion/contact sensors
- power plugs and energy meters
- media players/speakers
- air-quality sensors
- smart lights
- dishwasher/washing machine power monitoring
- homelab server/NAS
- Proxmox host(s)

## Design principle

Prefer boring, observable automations:

- every automation should have a clear trigger
- noisy alerts should be suppressed
- family-facing alerts should be actionable
- anything security-sensitive should stay private
